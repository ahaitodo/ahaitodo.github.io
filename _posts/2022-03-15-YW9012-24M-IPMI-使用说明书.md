YW9012-24M IPMI 使用说明书

一、功能说明

▲默认 IP 地址：192.168.0.188 

用户名：root 密码：admin 

▲设置 IP 

设置 IPMI 板的 IP，设置后会自动重启。

▲设置 MAC 

用于输入 IPMI 控制的 6 个主板的 MAC 地址，设置后可用 ipmitool 读取。设置后立即生效。

▲设置密码

更改 ipmitool 的 root 用户的密码，更改后重启主机后才能生效。

▲一键升级

使用 IPMI 升级工具，添加本地文件进行升级。

▲重启主机

重启 IPMI 系统。

▲增加 Windows7/10 工具

A、批量升级。 B、批量MAC 地址写入。 C、修改IP 网络地址

二、ipmitool 支持的命令

1） 命令 【读取开关机状态】

ipmitool -I lan -H <ip> -p <机器端口> -U <用户名> -P <密码> chassis power status 

输出："Chassis Power is on" 或者 "Chassis Power is off" 

示例：

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis power status 

2） 命令【开机】

ipmitool -I lan -H<ip> -p <机器端口> -U <用户名> -P <密码> chassis power on 

输出：

Chassis Power Control: Up/On 

示例：

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis power on 

3） 命令【正常关机】

ipmitool -I lan -H<ip> -p <机器端口> -U <用户名> -P <密码> chassis power soft 

输出：

Chassis Power Control: Soft 

示例：

 ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis power soft 

4） 命令【强制关机】

ipmitool -I lan -H<ip> -p <机器端口> -U <用户名> -P <密码> chassis power off 

输出：

Chassis Power Control: Down/Off 

示例：

 ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis power off 

5） 命令【强制重启】

ipmitool -I lan -H<ip>-p <机器端口> -U <用户名> -P <密码> chassis power reset 

输出：

Chassis Power Control: reset 

示例：

 ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis power reset 

6） 命令【正常重启】

ipmitool -I lan -H<ip>-p <机器端口> -U <用户名> -P <密码> chassis power cycle 

输出：

Chassis Power Control: cycle 

示例：

 ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis power cycle 

7） 命令【读取用户列表】

ipmitool -I lan -H<ip>-p <机器端口> -U <用户名> -P <密码> user list 

输出：输出用户列表

示例：

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin chassis user list 

8） 命令【更改密码】

ipmitool -I lan -H<ip> -p <机器端口> -U <用户> -P <密码> user set password<userid><新密码> 

输出：命令执行成功无输出

示例： 

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin user set password 2 test 

9） 命令【读取 IPMI 板的 MAC 地址】

ipmitool -H 192.168.0.188 -A NONE -p 6001 -I lan raw 0x3c 0xf0 

10） 命令【读取单个主板的 MAC 地址】

ipmitool -H 192.168.5.201 -A NONE -p 6001 -I lan raw 0x3c 0xf1 

11） 命令【获取时间】

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin sel time get 

输出：

05/01/2020 20:00:00 

12） 命令【修改时间】

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin sel time set "05/01/2020 11:43:07" 

输出：

05/01/2020 11:43:07 

示例：

如果 IPMI 的 IP 无法连通公网的 NTP 时间服务器，那么请定期检查并设置时间，以保证日志中的时

间准确。

13） 命令【日志】

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin sel elist 

这个是得到各种操作的日志信息。

14） 命令【IPMI 重启】

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin bmc reset warm 

输出：

Sent warm reset command to MC 

示例：

如果向某个端口发送命令失去响应，可尝试使用其它端口发送 IPMI 重启命令。

15） 命令【当前设备总功率】

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin raw 60 242 

输出结果如：0a 9b   十六进制转换成十进制即当前功率。

这个是得到 IPMI 当前的总的功率。

16） 命令【设置单台 MAC 地址】

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin raw 60 243 xx xx xx xx xx xx 

这个是设置单台机器的 MAC 地址，xx xx xx xx xx xx 6 个十进制数是 MAC 地址数。

17） 查看 IPMI 固件版本

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin raw 60 244 

查看当前IPMI 板子上的版本，表示的是 2.0.6 版本

18） 查看机箱风扇转速

ipmitool -I lan -H 192.168.0.188 -p 6001 -U root -P admin raw 60 250

输出结果如：32 47 32 47 32 47 32 47，则拿32 47 十六进制转换成十进制即当前转速。

19） IPMI 看门狗功能

IPMI 系统中设置了定时检测系统的看门狗，如果看门狗发现异常会自动重启 IPMI 系 统。从异常发生到

被检测出并重启完成恢复 ipmitool 的响应大概需要 120 秒。
